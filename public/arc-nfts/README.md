# Arc NFTs Images

Coloque as imagens dos NFTs da Arc Collection nesta pasta.

## Estrutura Esperada

```
public/
└── arc-nfts/
    ├── arc1.png
    ├── arc2.png
    ├── arc3.png
    ├── arc4.png
    └── arc5.png
```

## Requisitos das Imagens

- **Formato**: PNG, JPG, ou outro formato suportado por navegadores
- **Tamanho recomendado**: 512x512px ou 1024x1024px (quadrado)
- **Nomes**: arc1.png, arc2.png, arc3.png, arc4.png, arc5.png

## Nota

As imagens aqui são apenas para exibição no frontend. As URLs reais das imagens nos metadados JSON (tokenURI) devem apontar para onde você hospedou as imagens publicamente (GitHub, IPFS, servidor próprio, etc.).
